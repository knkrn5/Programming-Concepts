== Getting Started

=== Prerequisites

* Asciidoctor (for HTML/PDF): `asciidoctor`
* An editor with AsciiDoc preview (VS Code + Asciidoctor extension)

NOTE: If you plan to distribute printed docs, set `:doctype: book` and use Asciidoctor PDF.

TIP: Use attributes to keep project-wide values consistent (see top of document).

== Feature Comparison

|===
| Feature | Markdown | AsciiDoc

| Headings, lists, bold/italic
| ✅
| ✅

| Metadata (author, title)
| ❌
| ✅

| ❌
| ✅

| ❌

| ❌
| ✅

| ❌

| ❌
| ✅
